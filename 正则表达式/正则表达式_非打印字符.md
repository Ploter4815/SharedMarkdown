非打印字符是指计算机中字符确实存在，但不会显示或打印出来
1. `\cx`: 由 x 指明的**控制字符**
	- x 的值必须为 A-Z 或 a-z 之一，否则将 c 视为一个原义的 c 字符, `\` 此时作为转义字符
2. `\f`: **换页符** 
3. `\n`: **换行符**
4. `\r`: **回车符**
5. `\t`: **制表符**
6. `\v`: **垂直制表符**
7. `\s`: 任何**空白字符**
8. `\S`: 任何**非空白字符**

| 非打印字符 | 等价表达           |
| :----------: | :------------------: |
| \\cx       |                    |
| \\f        | \\x0c,\\cL         |
| \\n        | \\x0a,\\cJ         |
| \\r        | \\x0d,\\cM         |
| \\t        | \\x09,\\cI         |
| \\v        | \\x0b,\\cK         |
| \\s        | \[\\f\\n\\r\\t\\v] |
| \\S        | \[^\\f\\n\\r\\t\\v]                   |
