相同优先级的从左到右进行运算，不同优先级的运算先高后低。

| 运算符                 | 描述           |
| ---------------------- | -------------- |
| \\                     | 转义符         |
| (), (?:), (?=),\[\]    | 圆括号和方括号 |
| \*,+,?,{n},{n,},{n, m} | 限定符         |
| ^,$,\\任何元字符       | 位置和顺序     |
| \|                     | 替换，`或` 操作<br>字符具有高于替换运算符的优先级<br>"m\|food"匹配"m"/"food"<br>"(m\|f)ood"匹配"mood"/"food"               |
